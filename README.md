# 小鱼健康 - Harmony Health App

一个现代化的个人健康管理应用，帮助你追踪体重、BMI、健康状态和养生计划。

## 功能特性

### 📊 首页
- **体重 & BMI 追踪** - 实时计算 BMI 指数，显示健康等级
- **周健康状态** - 记录 4 个维度的健康指标：
  - 精力与睡眠
  - 神经与情绪
  - 消化与代谢
  - 骨骼与肌肉
- **养生计划打卡** - 每周追踪养生习惯完成情况
- **周备注** - 记录本周身体感受

### 📈 数据中心
- **BMI 趋势图** - 可视化体重指数变化
- **健康趋势分析** - 各维度健康状态的趋势图表
- **习惯达成率** - 环形图展示养生计划完成率
- **月度/年度视图** - 灵活的时间范围选择

### ⚙️ 设置
- **自定义健康记录项目** - 添加/编辑/删除健康指标
- **自定义养生计划** - 管理个人养生习惯
- **分类管理** - 组织健康指标的分类

## 技术栈

- **前端框架**: 原生 HTML5 + CSS3 + JavaScript
- **图表库**: Chart.js 4.4.1
- **存储**: 浏览器 LocalStorage（自动保存）
- **响应式设计**: 移动优先设计，完美适配各种屏幕

## 快速开始

### 在线使用
直接在浏览器中打开 `index.html` 文件即可使用。

### 本地部署
```bash
# 克隆仓库
git clone https://github.com/davisyu1003/harmony-health-app.git
cd harmony-health-app

# 使用任何本地服务器打开
# 方式1: Python
python -m http.server 8000

# 方式2: Node.js (http-server)
npx http-server

# 方式3: 直接打开文件
open index.html
```

然后在浏览器中访问 `http://localhost:8000`

## 使用指南

### 首页操作
1. **输入体重和身高** - 自动计算 BMI 指数
2. **选择周份** - 在顶部周条中切换不同周份
3. **记录健康状态** - 点击数字 1-4 记录各项健康指标
4. **打卡养生计划** - 点击 ✓ 或 ✗ 标记完成情况
5. **保存备注** - 记录本周身体感受

### 数据中心
- 查看 BMI 趋势和健康指标的可视化图表
- 切换月度/年度视图
- 使用左右箭头导航不同月份

### 设置
- 添加新的健康分类或养生计划
- 编辑现有项目名称
- 删除不需要的项目
